Api Request


Documentation.


Your request will be through the ISBN attribute and will have as a response a json object.


* Request form: res = requests.get('http://URLBOOKSSEARCHER/api/' + ISBN)

URLBOOKSSEARCHER will depend on the domain that this site has at the time (if it is running on your computer it will be "localhost: port" or "127.0.0.1:port").

ISBN is a string type variable. If the data is used literally instead of a variable then it must be enclosed in quotes.


Example of response to a real request to the local server:

res = requests.get('http://127.0.0.1:5000/api/' + '074349671X')
